# Dashboard Flow

SOPHIA's visual navigation is nested within a single, secure page container.

## Navigation Layout

* **Overview (Home)**: Displays stat metrics and recent focus checklists.
* **Calendar**: Syncs agenda grids and displays cognitive workload slots.
* **AI Command**: Connects to the agent command console.
* **Memory**: Displays knowledge nodes and semantic search options.
* **Notes**: Interactive markdown page editor.
* **Analytics**: Charts daily scores and load indices.
* **Settings**: Manages model configs and warning sliders.
